# BUILD PLAN — MarTech + AdTech Academy

## Objective
Build a standalone learning product that combines deep domain research + practical labs for MarTech and AdTech, delivered in Harvard-style case modules.

## Product shape
- **Format:** Guided learning app (Next.js App Router)
- **Audience:** Builders/operators moving into marketing systems and ad infrastructure
- **Learning model:** Theory + practical lab + Harvard case for every module
- **Progress model:** Module completion tracking in browser localStorage

## Applied techniques from prior academy builds
1. **Module-first architecture** (OpenClaw Academy): clear sequence, objectives, key terms, checkpoints.
2. **Interactive learner flow** (Redditech Academy spec): dedicated routes per module with navigation and progress tracking.
3. **Operator framing** (Redditech Labs style): every concept tied to execution, metrics, and tooling decisions.
4. **Case-led pedagogy**: situation framing, decision options, tradeoffs, execution plan, discussion prompts.
